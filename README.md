# 🔍 Semantic Search for Personal Messengers

Локальная система семантического поиска по истории переписок в Telegram. Проект позволяет искать сообщения по смыслу, а не по точному совпадению слов, а также автоматически распознает голосовые и видеосообщения. Программа имеет открытый исходный код и НЕ отсылает личные данные, поэтому полностью безопасен для использования.

## 🌟 Особенности
- **Семантический поиск:** Понимание контекста, синонимов и опечаток благодаря SOTA-моделям.
- **Двухэтапный поиск (Retrieval + Reranking):** Использование Bi-Encoder (`multilingual-e5-base`) для быстрой выборки и Cross-Encoder (`bge-reranker-v2-m3`) для точной сортировки результатов.
- **Распознавание медиа:** Автоматическая транскрибация голосовых сообщений и "кружочков" с помощью локальной модели **OpenAI Whisper**.
- **Векторная база данных:** Хранение и быстрый поиск эмбеддингов с помощью **ChromaDB**.
- **Веб-интерфейс:** Удобный и минималистичный UI на базе **Streamlit**.

## 🛠 Стек технологий
- **Python 3.9+**
- **ML & NLP:** Sentence-Transformers, OpenAI Whisper
- **Vector DB:** ChromaDB
- **Data Processing:** Pandas
- **UI:** Streamlit
